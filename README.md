# Cohort Sync

## Description
Local plugin for automatic cohort synchronization in Moodle. 
It updates cohort memberships based on configured rules or integrations.

## Features
- Automatic cohort synchronization
- Scheduled task support
- Custom rule-based processing
- Logging and error handling

## Requirements
- Moodle 4.x or later
- PHP 8.0 or later

## Installation
1. Copy the plugin folder to `/local/cohortsync`
2. Visit Site administration > Notifications
3. Follow the installation process

## Configuration
After installation, go to:
Site administration > Plugins > Local plugins > Cohort Sync

## Scheduled Tasks
The plugin uses Moodle scheduled tasks to process synchronization.
